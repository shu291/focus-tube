# 参考書ノート 📖

参考書と一緒に使う「間違い記録」アプリ。
**どのページのどの問題で・どこを間違えて・次はどう解くか**を写真つきで記録し、
忘れたころに解き直しをうながしてくれます。

## できること

### ✍️ 記録
- 参考書・ページ数・問題番号を指定して記録
- **どこで間違えた?(原因)** と **次はどう解く?(対策)** をメモ
- 📷 カメラ / 🖼 写真ライブラリから問題や答案の写真を複数枚添付(自動で圧縮)
- 単元・タグ付け、理解度(✕もう一度 / △あと少し / ⭕できた)

### 📋 一覧
- キーワード検索、本・状態での絞り込み
- 写真はタップで拡大表示
- 編集・削除

### 🔁 復習(解き直し)
- 「✕」「△」の問題を **1 → 3 → 7 → 14 → 30日** の間隔で自動出題
- 解き直したら ✕まだ / △あと少し / ⭕克服! で記録
- 出題予定日も一覧で確認できる

### 📚 本棚
- 参考書を教科別カラーで管理
- 本ごとの克服率をバーで表示

### 💾 データ保存
- 記録・写真はすべて**端末内(IndexedDB)に自動保存**。サーバーには送信されません
- JSONファイルへのエクスポート / インポートで機種変更・バックアップに対応

## 自動アップデートのしくみ
- このリポジトリは GitHub Actions による **GitHub Pages への自動デプロイ**が設定済みです
  (`.github/workflows/pages.yml`)。`main` にマージされると自動でサイトが更新されます。
- アプリ側は Service Worker が **HTMLをネットワーク優先**で取得するため、
  ユーザーは何もしなくても**次に開いたときに自動で最新版**になります(更新時はトースト通知)。
- 記録データはキャッシュとは別の IndexedDB にあるので、アップデートで消えません。

## 使い方
`study-notebook/index.html` を開くだけ。ホーム画面に追加すればPWAとして
オフラインでも使えます。
