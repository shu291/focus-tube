# 引き出しプランナー

引き出し収納のレイアウトを **実寸(mm)** でモデリングするアプリ。
メジャーで測った引き出しの内寸を登録し、しまいたい物を上から見た図に並べて
「入るか・どう並べるか」を事前にシミュレーションできる。

## 使い方

1. 「＋ 引き出し」で引き出しを登録(プリセットあり / 内寸 幅×奥行×高さを自由入力)
2. 「＋ アイテムを追加」でカタログから選ぶ
   - 筆記具・文房具・工具・キッチン・衛生・デジタル・裁縫・書類・その他の約180種
   - どのアイテムも追加時に **幅・奥行・高さを自分の実物に合わせて変更できる**
   - カタログにない物は「オリジナル登録」でマイカタログに保存
3. キャンバス上でドラッグして配置(5mmスナップ)。タップで選択して回転・複製・削除・サイズ変更
4. 自動チェック
   - ⛔ 赤枠: 他のアイテムとの重なり / 引き出しからのはみ出し
   - ⚠: アイテムの高さが引き出しの内寸(高さ)を超過
   - 面積使用率・アイテム数を常時表示

## 技術メモ

- 自己完結の単一 `index.html`(ビルド不要・依存なし)
- データは `localStorage`(キー: `drawerPlanner.v1`)に保存
- キャンバスは mm → px のスケーリング(画面幅にフィット、グリッド1マス = 50mm)
- Pointer Events によるドラッグ(タッチ/マウス両対応)
