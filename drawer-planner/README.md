# 引き出しプランナー

引き出し収納のレイアウトを **実寸(mm)** でモデリングするアプリ。
メジャーで測った引き出しの内寸を登録し、しまいたい物を上から見た図に並べて
「入るか・どう並べるか」を事前にシミュレーションできる。

## 使い方

1. 「＋ 引き出し」で引き出しを登録(プリセットあり / 内寸 幅×奥行×高さを自由入力)
2. 「＋ アイテムを追加」でカタログから選ぶ
   - 筆記具・文房具・工具・キッチン・衛生・デジタル・裁縫・書類・その他の約180種
   - どのアイテムも追加時に **幅・奥行・高さを自分の実物に合わせて変更できる**
   - カタログにない物は「オリジナル登録」でマイカタログに保存
3. キャンバス上でドラッグして配置(5mmスナップ)。タップで選択して回転・複製・削除・サイズ変更
4. 自動チェック
   - ⛔ 赤枠: 他のアイテムとの重なり / 引き出しからのはみ出し
   - ⚠: アイテムの高さが引き出しの内寸(高さ)を超過
   - 面積使用率・アイテム数を常時表示
5. 「🖨 3D」で、配置に合わせた **仕切りトレー(オーガナイザー)を3Dプリント用に書き出す**

## 3Dプリント(仕切りトレー書き出し)

配置したアイテムにぴったり合う仕切り付きの収納トレーを生成し、バイナリ **STL** で
ダウンロードできる。スライサー(Cura / PrusaSlicer / Bambu Studio 等)にそのまま
読み込んで印刷できる。ライブラリ非依存で、モーダル内にアイソメトリックの
3Dプレビューを表示。

- **ベースの形** — 「アイテムに合わせる」(必要最小サイズ)/「引き出しサイズ」
  (外周壁つきの引き出しにそのまま入るインサート)
- **仕切りの高さ** — 「そろえる」(縁を水平に)/「アイテムごと」(各物の高さに合わせる)
- **壁の厚み・底の厚み・すき間**(実物とのゆとり)を mm 単位で調整。プレビューと
  出力サイズが即時更新される
- **作る範囲** — アイテムを選択しているときは「選択中のアイテムだけ」を選べ、
  その物専用のホルダーを1つだけ書き出せる
- 単位は mm。STL の 1 単位 = 1mm なので寸法どおりに印刷される

### STLの作り(実装メモ)

トレーは **軸並行ボックスの集合**として生成する ―― 床スラブ + アイテムごとの
囲い壁(4枚)+(引き出しサイズ時)外周壁。各ボックスは 12 三角形で、外向き CCW の
法線つき。隣り合う仕切りの壁は重なって共有壁になる(閉じたソリッドの重なりは
スライサーが union するため印刷上問題ない)。座標は原点(0,0,0)基準に平行移動。

## 技術メモ

- 自己完結の単一 `index.html`(ビルド不要・依存なし)
- データは `localStorage`(キー: `drawerPlanner.v1`)に保存
- キャンバスは mm → px のスケーリング(画面幅にフィット、グリッド1マス = 50mm)
- Pointer Events によるドラッグ(タッチ/マウス両対応)
- STL 生成は `DataView` によるバイナリ書き出し、プレビューは 2D Canvas の
  アイソメ投影(いずれも外部依存なし)
