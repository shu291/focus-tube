# おうち在庫 📦

家族で共有する、家庭の消耗品ストック管理アプリです。
GitHub Pages 上で動く軽量な PWA（インストールできるWebアプリ）で、データは常に自動保存されます。

➡️ 公開URL: `https://<あなたのユーザー名>.github.io/focus-tube/inventory/`

## できること

- 📷 **写真と文字で登録** — カメラで撮る / 写真をえらぶ。名前・置き場所・種類・単位・メモも入力。
- ➕➖ **個数をワンタップで増減** — 「＋」「－」ボタンと「使った −1」ボタンで画面からかんたん操作。
- 🏷 **種類・場所で分類** — 場所別／種類別にまとめて表示。チップで絞り込み。
- 🔀 **いろいろな並べ替え** — 名前順（あいうえお/ABC）、よく使う順（使用頻度）、在庫が少ない順、最近さわった順 など。
- ⚠️ **残りわずかの警告** — アイテムごとに目安数を設定。少なくなると色が変わり、まとめて表示できます。
- ☁️ **家族で共有** — クラウド同期を設定すると、家族みんなが同じデータをリアルタイムに見て編集できます（何年でも継続）。
- 💾 **バックアップ** — JSONファイルで書き出し／読み込み。
- 📴 **オフライン対応** — 一度開けばネットがなくても起動。クラウド同期はオンライン時に自動反映。

## データの保存について

| 使い方 | 保存先 | 共有 |
|--------|--------|------|
| そのまま使う | この端末（ブラウザ内） | × 端末ごと |
| クラウド同期をONにする | Firebase（無料） | ◎ 家族みんなで同じデータ |

## 家族で共有する設定（クラウド同期）

無料の Firebase Firestore を使います。一度だけ次の準備をします。

1. [Firebase コンソール](https://console.firebase.google.com/) で新しいプロジェクトを作成
2. 左メニューの **Firestore Database** を作成（テストモードでも可）
3. **プロジェクトの設定 → マイアプリ → ウェブ（`</>`）** を追加
4. 表示される `firebaseConfig`（`apiKey` などが書かれた部分）をコピー
5. アプリの **⚙️設定** を開き、`firebaseConfig` を貼り付け
6. **家族の合言葉**（例: `yamada-family`）を入力して「共有を開始」

家族の各端末で **同じ firebaseConfig と同じ合言葉** を入力すれば、全員が同じ在庫データを共有・編集できます。

### Firestore セキュリティルールについて

家族だけで使う合言葉ベースの簡易共有です。合言葉が他人に知られないようにしてください。
より厳密に保護したい場合は、Firebase Authentication と組み合わせてルールを設定してください。

写真は端末側で縮小（最大720px・JPEG）してから保存するため、無料枠でも十分に使えます。

## 技術メモ

- 依存ライブラリなしの単一HTML（`index.html`）。クラウド同期時のみ Firebase compat SDK を CDN から読み込み。
- `manifest.webmanifest` / `sw.js` でPWA・オフライン対応。
- 同じリポジトリの GitHub Pages ワークフロー（`.github/workflows/pages.yml`）でそのまま公開されます。
