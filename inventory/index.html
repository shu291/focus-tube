☁️ 家族で共有（クラウド同期）

家族みんなで同じデータを見て、何年でも編集できるようにします。無料のFirebaseを使います。
  1. Firebaseコンソールで新しいプロジェクトを作成
  2. 「Firestore Database」を作成（本番/テストどちらでも可）
  3. プロジェクト設定 → マイアプリ →「ウェブ(</>)」を追加
  4. 表示される firebaseConfig の中身を下に貼り付け
{ } の中身ごと、または apiKey: "..." の形のまま貼ってOK。
同じ合言葉を入れた人どうしで在庫データを共有します。
