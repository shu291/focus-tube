# 🍳 きょうの献立ルーレット

「今日なに作ろう…」を解決する、献立ランダムメーカーです。
難易度を選んでルーレットを回すと、**ジャンル・主食・メイン食材・調理法** の4つが一斉に決まり、
Gemini AI が条件に合った具体的なレシピ(料理名・材料・作り方)を提案します。

## つかいかた

1. **人数** と **難易度(簡単 / 普通 / 凝った料理)** を選ぶ
2. 「🎰 4つ一斉に回す！」でルーレットを回す
3. 決まった組み合わせで「レシピを見る」
4. AI が具体的なレシピを生成(キー未設定なら検索リンクで代替)

## Gemini API キーについて

- AI レシピ機能を使うには、無料の Gemini API キーが必要です。
  [Google AI Studio](https://aistudio.google.com/app/apikey) で取得できます。
- キーはアプリの ⚙️ 設定から入力します。
- キーは **利用者のブラウザ(localStorage)内にのみ保存** され、サーバーには送信されません
  (Google の API へのリクエストを除く)。
- キーを設定しなくても、Google検索・クックパッド・楽天レシピ・YouTube の
  検索リンクで料理を探せます。

> ⚠️ このアプリは静的サイトのため、API キーはブラウザから直接 Google の API へ送られます。
> 公開リポジトリにご自身のキーを書き込まないでください(各ユーザーが自分のキーを入力する設計です)。

## 公開方法 (GitHub Pages)

1. リポジトリの **Settings → Pages** で公開ブランチを選択
2. 公開後、`https://<ユーザー名>.github.io/<リポジトリ名>/recipe-randomizer/` でアクセスできます

## 構成

- `index.html` … 画面(スタート / ルーレット / レシピ)
- `style.css` … 明るめのデザイン
- `app.js` … ルーレットのロジックと Gemini API 連携

外部ライブラリ不要・ビルド不要のバニラ構成です。
