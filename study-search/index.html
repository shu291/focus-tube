⚙️ APIキー設定

2つとも無料で発行できます。キーはこの端末の中にだけ保存されます。
→ Google AI Studio でキーを作る
→ Google Cloud で YouTube Data API v3 を有効化してキーを作る
キーが他人に使われないように、Google Cloud / AI Studio 側で「HTTPリファラー制限」をかけておくと安心です。
YouTube検索の無料枠はおよそ1日100回分です(検索1回=100ユニット/10,000)。
