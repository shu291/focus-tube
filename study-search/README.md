# 勉強サーチ 🔍

**Gemini が検索ワードを判定してくれる、勉強専用の YouTube 検索アプリ。**
「数学 二次関数」のような勉強のキーワードは検索できますが、
「フォートナイト 実況」のようなゲーム・エンタメ系のキーワードは ⛔ ブロックされて検索できません。

## しくみ

```
検索ワード入力
   ↓
① Gemini API が「勉強に関係あるか」を判定
   ├─ ✕ 関係ない → ⛔ ブロック(理由を表示。YouTube検索はしない)
   └─ ○ 関係ある ↓
② YouTube Data API で動画を検索(セーフサーチON)
   ↓
③ 検索結果の動画一覧をもう一度 Gemini がチェックして、
   紛れ込んだゲーム実況・エンタメ動画を除外
   ↓
アプリ内プレイヤーで再生(おすすめ・コメントなしで寄り道できない)
```

- 一度判定したキーワードは端末内にキャッシュされるので、2回目からは判定なしですぐ検索できます
- 「許可して」「これは勉強です」のようにキーワードでお願いしても判定はゆるまない実装です

## できること

- 🔍 勉強に関係するキーワードだけ YouTube 検索(関係ないものは理由つきでブロック)
- 🧹 検索結果からも勉強に関係ない動画を自動で除外
- ▶ アプリ内プレイヤーで再生(おすすめ動画・コメント欄が出ないので集中できる)
- 🕐 最近の検索(許可されたものだけ)をワンタップで再検索
- 📱 ホーム画面に追加すればPWAとして起動できます

## 使うために必要なもの(どちらも無料)

このアプリはサーバーを持たない静的ページなので、自分のAPIキーを2つ用意して
アプリの「⚙️ APIキー設定」に入れます。**キーは端末のブラウザ内(localStorage)にだけ保存され、
GoogleのAPI呼び出し以外にどこにも送信されません。**

### 1. Gemini APIキー(検索ワードの判定用)

1. [Google AI Studio](https://aistudio.google.com/app/apikey) を開いて Google アカウントでログイン
2. 「APIキーを作成」を押してキーをコピー

### 2. YouTube APIキー(動画検索用)

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクトを作成(無料)
2. [YouTube Data API v3](https://console.cloud.google.com/apis/library/youtube.googleapis.com) を開いて「有効にする」
3. 「APIとサービス → 認証情報 → 認証情報を作成 → APIキー」でキーをコピー

### おすすめの安全設定

- 2つのキーとも「**HTTPリファラー制限**」で自分のサイト(例: `https://◯◯.github.io/*`)だけに制限しておくと、キーが漏れても他人に使われにくくなります
- YouTube 検索の無料枠は **1日あたり約100回**です(検索1回 = 100ユニット / 日次上限10,000ユニット)
- Gemini は無料枠内で使えます(判定は1検索につき最大2回・ごく短い呼び出し)

## 使い方

`study-search/index.html` を開くだけ。初回に「⚙️ APIキー設定」から2つのキーを保存すれば、
あとは検索するだけです。ホーム画面に追加すればアプリのように使えます。
