# ミニ単語帳メーカー 📇

英単語・日本語訳・メモを入力して、好きな用紙サイズ・レイアウトで
**印刷用のミニ単語帳**を作るツール。プリンターでA4などに印刷して、
切る／折る／両面印刷することで小さな単語帳が完成します。

## 使い方
1. **単語を入れる** — 1つずつ入力するか、まとめて貼り付け（`単語, 訳` または `単語[tab]訳[tab]メモ`）。
2. **サイズ・レイアウトを選ぶ**
   - 用紙サイズ：A4 / A5 / A6 / B5 / B6 / L判
   - 向き：たて / よこ
   - 列数・行数・余白・隙間・文字サイズを自由に調整
   - 作り方：
     - **切って使う（片面）** … グリッドに並べて印刷 → 線で切る
     - **2つ折り** … 「単語｜訳」を並べて印刷 → 中央で折る
     - **両面印刷** … 表＝単語 / 裏＝訳 のページを生成（長辺とじ）
3. **プレビューを作る → 印刷** — 印刷ダイアログで用紙を合わせ、余白を「なし／最小」に。

## 特徴
- インストール不要、単一HTMLファイルで完結
- データはどこにも送信されません（すべてブラウザ内で処理）
- 番号付け、表示項目（単語／訳／メモ）の切り替えに対応

`flashcard-maker/index.html` をブラウザで開くだけで使えます。
